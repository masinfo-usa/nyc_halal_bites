I build a website for a food truck called Mezquite Valley.

The frontend is developed using React.js with the Material UI framework, providing a clean, modern, and responsive design that automatically adjusts to different screen sizes — large, medium, and mobile.

The backend integrates Square’s Payment API to securely process customer payments and Square’s Order API to send placed orders directly to the Square Terminal used at the food truck.

The orders, customers info is stored in MongoDB.

The website is now active and can be viewed here:
https://mezquite-valley.com
